DdMd::LammpsConfigIo Class Reference
ConfigIos

Detailed Description

Lammps data file format for configuration files.

LammpsConfigIo is a ConfigIo that reads and writes the LAMMPS data file format for configuration files. Atoms are output with ids in sequential order.

MaskPolicy  maskPolicy 
)
virtual

Read configuration file.

Call on all processors. File is used and must be open only on master. This routine opens and reads a file on the master, and distributes atom data among the processors.

Precondition
There are no atoms, ghosts, or groups.
AtomStorage is set for scaled / generalized coordinates
Postcondition
atomic coordinates are scaled / generalized
there are no ghosts
Parameters
fileinput file stream
maskPolicyMaskPolicy to be used in setting atom masks
