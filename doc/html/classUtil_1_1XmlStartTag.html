Util::XmlStartTag Class Reference
Util namespace » Xml Tag Parsers

Detailed Description

Parser for an XML start tag.

Usage:

XmlStartTag tag;
XmlAttribute attribute;
std::string line;
tag.matchLabel(line, 0);
while (matchAttribute(attribute)) {
// process attribute;
}
tag.finish();
