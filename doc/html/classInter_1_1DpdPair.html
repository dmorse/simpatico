double Inter::DpdPair::forceOverR ( double  rsq,
int  i,
int  j 
) const
inline

Returns ratio of scalar pair interaction force to pair separation.

Multiply this quantity by the components of the separation vector to obtain the force vector. A positive value for the return value represents a repulsive force between a pair of particles.

Precondition: The square separation rsq must be less than cutoffSq. If rsq > cutoffSq, the return value is undefined (i.e., wrong). Usage: Test for rsq < cutoffSq before calling this function

if (rsq < interaction.cutoffSq(i, j)) {
f = forceOverR(rsq, i, j);
.....
}
Parameters
rsqsquare of distance between particles
itype of particle 1
jtype of particle 2
Returns
force divided by distance
