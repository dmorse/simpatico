McMd::RingOctaRebridgeMove Class Reference
Monte Carlo Moves

Detailed Description

Molecule rebridging move for a Ring species.

The exchange of positions of two closely approaching atoms is attempted. Each atom is bonded to two other atoms. The move requires that the 6 atoms form an octahedron in which the edge lengths satisfying certain distance criterions, before the exchange is attempted.
